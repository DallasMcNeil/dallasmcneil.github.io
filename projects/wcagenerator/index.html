Instructions

1. Add competition data
Get data from https://www.worldcubeassociation.org/api/v0/competitions/YourCompetitionID/wcif/ Replace YourCompetitionID with your competition name

2. Select template

3. Adjust settings

  • Set background image
  • Page margins 3%
  • Include staffing in schedules
  • Include stations in schedules

4. 'Generate' and preview the badges

5 'Print' when ready and use these settings in the print dialog

  • Layout: Landscape
  • More Settings
    • Paper Size: A4
    • Margins: None
    • Background Graphics: On
